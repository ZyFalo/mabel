# admin-brand-skin вҖ” Mockups ASCII

Mockups canГіnicos para guiar la implementaciГіn del frontend-developer. Densidad operativa, no asimetrГӯa de chat. Todos los textos en espaГұol con tildes correctas. Tokens entre llaves `{}` corresponden a `index.css`.

Leyenda:

- `в–Ҳ` bloque sГіlido (rojo brand, ink oscuro)
- `в–‘` superficie cГЎlida (ink-50, mabel-50)
- `в”Ӯ в”Җ в”Ң в”җ в”” в”ҳ в”ң в”Ө в”¬ в”ҙ в”ј` bordes de tabla / wrapper
- `в—Ҹ` punto threshold o estado activo
- `[texto]` botГіn
- `вҖ№textoвҖә` pill o chip
